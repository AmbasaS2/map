# 🗺️ 지도

**지도는 현재 RP를 읽어 장소, 인물의 위치, 퀘스트와 단서를 지도 위에 정리해주는 SillyTavern 확장입니다.**

---

## ⚙️ 확장 설정

### 🪄 지도 테마

| 테마 | 설명 |
|---|---|
| **Marauder's Map (HP AU)** | 양피지와 잉크, 발자국으로 펼쳐지는 마법 지도 테마입니다. 판타지 AU나 중세 배경에도 잘 어울립니다. |
| **Location tracker (Modern AU)** | 밝은 위치 추적기 화면으로 장소와 인물의 위치 신호를 확인하는 현대 AU 테마입니다. |

### 🔤 글씨 크기

지도 안의 글씨 크기를 조절합니다.

### 🔌 연결 프로필

지도 생성에 사용할 API를 선택합니다.

- **메인 API** — 현재 SillyTavern에서 사용 중인 연결을 그대로 사용합니다.
- **별도 연결 프로필** — Connection Manager에 저장한 프로필을 지도 전용으로 사용합니다.

### 🐞 디버그 로그

지도 생성이나 저장이 제대로 되지 않을 때 현재 페이지에서 발생한 최근 요청 기록을 확인합니다. 로그를 복사하거나 비울 수 있으며, 브라우저 저장소에는 남지 않고 새로고침하면 사라집니다.

---


## 🗺️ 지도 열기

확장 메뉴의 **지도**를 누른 뒤 첫 화면의 주문 또는 **위치 추적 활성화** 버튼을 누릅니다.

- 현재 채팅에 저장된 지도가 있으면 기존 지도를 다시 펼칩니다.
- 저장된 지도가 없으면 최근 RP를 읽어 새 지도를 만듭니다.
- 지도와 직전 지도는 채팅별로 저장됩니다.

**Mischief Managed.** 또는 `×`는 지도 창을 닫는 버튼입니다. 지도 생성 중 창만 닫으면 생성은 계속 진행되며, 완료된 결과는 같은 채팅에 저장됩니다.

채팅을 이동하거나 확장을 끄면 이전 채팅에서 늦게 도착한 결과가 현재 지도에 저장되지 않도록 취소 처리됩니다.

---

## 📍 지도 화면

### 지도판

| 표시 | 기능 |
|---|---|
| **장소 노드** | 장소의 현재 분위기, 주변 인물과 퀘스트를 엽니다. |
| **장소 옆 숫자** | 해당 장소에서 확인할 수 있는 퀘스트 수입니다. |
| **👣 발자국 / 위치 신호** | 지도에 포착된 인물의 현재 상태를 확인합니다. |
| **지도판 아래 핸들** | 위아래로 드래그해 지도판의 높이를 조절합니다. |

지도 위쪽에는 현재 구역, 시간대와 장면 요약이 표시됩니다.

### 상단 버튼

| 버튼 | 기능 |
|---|---|
| `🔄` | 최근 RP를 다시 읽어 전체 지도를 새로 만듭니다. 새로고침 전 지도는 직전 지도에 보관됩니다. |
| `↩️` | 현재 채팅의 직전 지도로 되돌아갑니다. |
| `🔎` | 현재 RP에 어울리는 장소를 검색합니다. |
| `📓` | 반영·수집한 정보와 추적 중인 인물을 관리하는 수첩을 엽니다. |
| `🧠` | 현재 지도에 저장된 장소, 퀘스트와 반영 상태를 요약해서 봅니다. |

---

## 🏛️ 장소

장소 노드를 누르면 오른쪽에 장소 정보가 열립니다.

| 확인할 수 있는 내용 | 설명 |
|---|---|
| **현재 상황** | 공간의 분위기와 지금 벌어지고 있는 움직임을 보여줍니다. |
| **주변 인물** | 현재 이 장소에 있는 인물이나 무리를 보여줍니다. |
| **퀘스트** | 이 장소에서 이어갈 수 있는 사건과 예상 보상을 보여줍니다. |

장소 카드의 `🔄`는 **선택한 장소만 다시 확인**합니다. 전체 지도를 새로 만드는 버튼이 아닙니다.

`📓 반영`에서는 장소 전체와 퀘스트를 함께 반영하거나, 원하는 퀘스트만 따로 선택할 수 있습니다.

---

## ✨ 퀘스트

지도는 현재 장면에서 자연스럽게 이어갈 수 있는 퀘스트를 제안합니다.

퀘스트에는 다음 내용이 표시됩니다.

- 퀘스트 내용과 현재 상황
- 퀘스트가 연결된 장소
- 진행하면서 열릴 수 있는 단서, 관계 변화나 선택지
- 예상 보상

퀘스트는 자동으로 RP에 시작되지 않습니다. 원하는 퀘스트를 직접 반영했을 때만 다음 답변에 참고됩니다.

---

## 👣 인물 위치 신호

지도 위의 발자국 또는 위치 신호를 누르면 해당 인물의 현재 상태를 확인합니다.

| 확인할 수 있는 내용 | 예시 |
|---|---|
| **기본 정보** | 이름, 성별, 나이, 현재 위치와 기분 |
| **관계** | 유저와의 관계 및 친밀도, 최근 마주친 순간 |
| **현재 상태** | 지금 하고 있는 행동과 생각 |
| **인벤토리** | 현재 가지고 있는 물건이나 단서 |
| **따라갈 만한 단서** | 이 인물로 이어갈 수 있는 장면 후보 |

`👁 추적`을 누르면 수첩의 추적 목록에 추가됩니다. `📓 반영`에서는 인물의 현재 상태와 생각을 다음 답변에 반영하거나 수첩에만 수집할 수 있습니다.

---

## 🔎 장소 검색

`🔎`에서는 현재 RP의 세계관과 장면에 어울리는 장소를 찾습니다.

빠른 검색으로 **맛집, 데이트 명소, 요즘 핫플레이스, 산책 코스**를 고르거나 원하는 조건을 직접 입력할 수 있습니다.

검색 결과에는 다음 내용이 표시될 수 있습니다.

- 장소 이름과 위치
- 짧은 소개와 미니 리뷰
- 유명한 이유와 시그니처
- 예상 시나리오와 가능한 장면

마음에 드는 장소는 바로 반영하거나 `📓` 수첩에 수집할 수 있습니다. `🗑️`를 누르면 해당 검색 결과와 연결된 수집·반영 설정도 함께 삭제됩니다.

---

## 👁 인물 추적

인물 정보에서 `👁 추적`을 누르면 수첩에서 이후 위치와 동선을 다시 확인할 수 있습니다.

| 기능 | 설명 |
|---|---|
| **지금 어디 있지?** | 현재 RP를 다시 읽어 인물의 위치와 행동을 갱신합니다. |
| **이전 동선** | 이전에 확인한 위치와 행동 기록을 펼쳐봅니다. |
| **반영 버튼** | 인물의 최근 동선과 현재 상태를 다음 답변에 반영합니다. |
| **추적 해제** | 추적 목록에서 인물을 삭제합니다. |

---

## 📓 반영과 수첩

장소, 퀘스트, 인물과 검색 장소는 아래 방식으로 관리합니다.

| 버튼 | 기능 |
|---|---|
| `🪄` | 선택한 정보를 다음 답변부터 배경 정보로 반영합니다. |
| `👤` | 캐릭터가 해당 정보를 먼저 알아차리도록 반영합니다. |
| `🧍` | 유저가 해당 정보를 먼저 알아차리도록 반영합니다. |
| `📓` | RP에는 반영하지 않고 수첩에만 저장합니다. |
| `🗑️` | 반영 설정만 해제합니다. 지도에 표시된 장소나 퀘스트 자체는 삭제되지 않습니다. |

반영을 선택한 정보만 SillyTavern의 확장 프롬프트를 통해 다음 AI 답변에 참고됩니다. 입력창에 문장이 자동으로 붙지는 않습니다.

수첩에서는 다음 정보를 한곳에서 관리합니다.

- 반영하거나 수집한 장소와 퀘스트
- 반영하거나 수집한 인물 정보
- 추적 중인 인물과 이전 동선
- 수집한 장소 검색 결과

수첩에 저장한 항목은 나중에 반영 방식을 바꾸거나 삭제할 수 있습니다.

## 🧠 저장되는 정보

| 구분 | 저장 범위 |
|---|---|
| **현재 지도와 직전 지도** | 현재 채팅에만 저장됩니다. 다른 채팅의 지도와 섞이지 않습니다. |
| **수첩과 추적 정보** | 같은 캐릭터의 공용 수첩에 저장되어 다른 채팅에서도 이어서 볼 수 있습니다. |
| **지도 메모리 `🧠`** | 현재 지도 이름, 생성 시각, 장소, 퀘스트와 반영 상태를 요약해서 보여줍니다. |

지도 확장은 `localStorage`나 `IndexedDB`에 지도 데이터나 디버그 로그를 저장하지 않습니다.

전체 지도를 새로고침하면 새 지도 바로 전 상태가 `↩️` 직전 지도에 저장됩니다. 직전 지도는 현재 채팅에 하나만 유지됩니다.

## 🤖 AI가 필요한 기능

| 기능 | AI 사용 | 설명 |
|---|---|---|
| **저장된 지도 열기** | 사용하지 않음 | 현재 채팅에 저장된 지도를 그대로 불러옵니다. |
| **새 지도 생성** | 사용 | 저장된 지도가 없을 때 최근 RP를 읽어 지도를 만듭니다. |
| **전체 지도 새로고침 `🔄`** | 사용 | 최근 RP를 다시 읽어 장소, 인물 위치 신호와 퀘스트를 새로 만듭니다. |
| **장소 카드 열기** | 사용하지 않음 | 지도에 저장된 장소 정보를 그대로 보여줍니다. |
| **장소 새로고침 `🔄`** | 사용 | 선택한 장소의 상황, 주변 인물과 퀘스트를 다시 확인합니다. |
| **장소 검색 `🔎`** | 사용 | 입력한 조건과 현재 세계관에 맞는 장소를 생성합니다. |
| **발자국·위치 신호 확인** | 처음 확인할 때 사용 | 한 번 생성한 인물 정보는 저장되며, 다시 열 때는 저장된 내용을 사용합니다. |
| **추적 인물의 ‘지금 어디 있지?’** | 사용 | 최근 RP를 읽어 해당 인물의 위치와 행동을 갱신합니다. |
| **반영·수집·추적 설정 변경** | 사용하지 않음 | 저장 방식이나 다음 답변 반영 여부만 바꿉니다. |


---

## ⚠️ 참고사항

- 지도 내용은 현재 RP와 캐릭터 정보를 바탕으로 생성되며, RP에 직접 나오지 않은 내용은 AI가 추정하거나 새로 만들어 추가할 수 있습니다.
- 지도 생성 중 창을 닫아도 생성은 계속됩니다. 생성을 중지하려면 현재 캐릭터와의 채팅방을 나가 다른 채팅으로 이동해야 합니다.
- 전체 지도 또는 장소 정보를 읽는 중에는 다른 새로고침 요청을 동시에 시작하지 않습니다.
